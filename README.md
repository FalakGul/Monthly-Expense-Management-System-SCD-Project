# Monthly Expense Management System SCD Project

A reconstructed and significantly enhanced version of the original Monthly Expenses project developed as part of the **Software Construction and Development** course.

---

# Project Overview

This project was completed as a course project for **Software Construction and Development**.

The original open-source project was a simple **Vehicle Expense Calculator** that allowed users to calculate the total cost of vehicle-related expenses.

This project reconstructs and extends that application into a **complete Monthly Expense Management System** by redesigning the software architecture, implementing CRUD functionality, improving the user interface, strengthening error handling, optimizing the codebase, and applying software construction principles.

The primary objective was not only to improve the existing application but also to demonstrate software maintenance, refactoring, defensive programming, documentation, and feature enhancement techniques.

# Transformation

## Original Application

The original project mainly functioned as a calculator for vehicle expenses.

Users could:

- Enter vehicle-related expenses
- Calculate the total amount

The application did not provide expense management capabilities such as storing records, editing entries, filtering, or maintaining history.

---

## My Enhanced Application

The application has been transformed into a complete **Monthly Expense Management System**.

Users can now:

- Add expenses
- Edit expenses
- Delete expenses
- View all expenses
- Search expenses
- Filter expenses
- View dashboard statistics
- Track total monthly spending
- View filtered totals
- Manage expense history

This makes the application suitable for managing day-to-day personal expenses instead of only calculating vehicle costs.
